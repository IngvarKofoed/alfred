34. Fixed a Gemini serialization bug that killed conversations once a tool returned an array. `asRecord` (packages/agent-core/src/providers/gemini.ts), used to coerce a tool result into Gemini's `functionResponse.response` (a proto Struct, which must be a JSON object), tested only `typeof value === 'object'` — but `typeof [] === 'object'`, so an array result (e.g. the browser `get_links` / `list_tabs` tools) was sent as a bare list and the API rejected the entire request ("Proto field is not repeating, cannot start list"), failing the run and every subsequent turn in that conversation.
